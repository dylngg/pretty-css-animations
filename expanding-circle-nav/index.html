Pure CSS Expanding Nav Circle

A simple example of a expanding circle nav that only uses css.

The nav works by using the ~general sibling selector, along with a checkbox that has a for="" toggle on the nav button to toggle whether the nav opens. The rest are just some divs that are hooked into the :checked detector: 

input[type=checkbox]:checked ~ .nav-content {
    /* Magic things happen here. */
}
